ATLAS 1
— life OS wireframes
[renameable]

Four directions, same shell

# Brief: a desktop life-OS (calendar, health, work, hobbies, AI projects, home). Plugin-style modules. Renameable per user. Wireframes, so: pencil-sketch feel, b&w + small accent pops, mono + handwritten pairing. Each direction shows onboarding → home → a module or two. Flip between them up top ↑

# What to look for: how dense each one feels, how AI and the command bar are treated, how navigation works, and how the "this is MY system, I named it" vibe is expressed. Click tweaks bottom-right to fiddle with the project name, theme, and density.

wireframe system
· fonts: JetBrains Mono + Caveat
· palette: ink on paper + tomato + blue + yellow
· rules: dashed = placeholder, solid = real UI, filled = active
· annotations in tomato caveat
· all boxes slightly rotated, hand feel
